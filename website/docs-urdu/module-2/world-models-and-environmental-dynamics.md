# دنیا کے ماڈلز اور ماحولیاتی ڈائی نامکس

## تعارف

دنیا کے ماڈلز فزکل ای آئی میں ایک بنیادی پیراڈائم کی نمائندگی کرتے ہیں جہاں جسمانی ایجنٹس ماحولیاتی ڈائی نامکس، جسمانی قوانین، اور سسٹم انٹرایکشن کی اندرونی نمائندگیاں سیکھتے ہیں۔ روایتی ماڈل فری نقطہ نظر کے برعکس جو صرف کوشش اور غلطی کے سیکھنے پر انحصار کرتا ہے، دنیا کے ماڈلز ایجنٹس کو اپنے جسمانی ماحول کے اندرونی نمائندگیوں کے اندر سیمولیٹ، پیش گوئی، اور پلین کرنے کی اجازت دیتے ہیں۔ ہیومنوائڈ روبوٹکس کے لیے، دنیا کے ماڈلز خاص طور پر اہم ہیں کیونکہ وہ محفوظ ایکسپلوریشن، کارآمد سیکھنا، اور پیچیدہ، متحرک ماحول میں مستحکم رویہ کو فعال کرتے ہیں جہاں حقیقی دنیا کے تجربات خطرناک یا مہنگے ہو سکتے ہیں۔

جسمانی نظاموں میں دنیا کے ماڈلز کی تصور سادہ پیش گوئی کے ماڈلنگ سے آگے بڑھ کر جسمانی تعاملات کی جاری قدرت کو قبضہ کرتا ہے، حالت کے تخمینے میں عدم یقینی کو سنبھالتا ہے، اور حقیقت کے وقت کام کے لیے کمپیوٹیشنل کارآمدگی برقرار رکھتا ہے۔ چیلنج درست ماڈلز سیکھنے میں ہے جو مختلف ماحولیاتی حالات میں جنرلائز کر سکیں جبکہ جسمانی تعامل کے لیے ضروری وفاداری برقرار رکھیں۔

## جسمانی نظاموں میں دنیا کے ماڈلز کی نظریاتی بنیادیں

### سبب کی ساخت سیکھنا

جسمانی نظاموں میں غنی سبب کی ساخت ہوتی ہے جسے دنیا کے ماڈلز قبضہ کرنا چاہیے تاکہ درست پیش گوئی اور پلیننگ کو فعال کیا جا سکے۔ جسمانی ماحول میں سبب کے تعلقات کا تعین قوانین کے ساتھ سٹوکاسٹک پرٹربیشنز کے ساتھ ہوتا ہے:

```
P(s_{t+1} | do(a_t), s_t) = f_dynamics(s_t, a_t, θ_physics) + noise
```

جہاں `do(a_t)` ایکشن `a_t` لینے کی مداخلت کی نمائندگی کرتا ہے، اور `f_dynamics` بنیادی جسمانی ڈائی نامکس کی نمائندگی کرتا ہے جس کی `θ_physics` کے ذریعے پیرامیٹرائزیشن کی گئی ہے۔ دنیا کے ماڈلز کو یہ سیکھنا چاہیے کہ تعلق اور سبب کے درمیان فرق کریں تاکہ منصوبہ بندی کے دوران غلط پیش گوئیوں سے گریز کیا جا سکے۔

جسمانی ماحول کی سبب گراف کی ساخت میں شامل ہے:
- حالت کی انحیاسیت: `s_t → s_{t+1}`
- ایکشن-حالت کے تعلقات: `a_t → s_{t+1}`
- ماحولیاتی عوامل: `e_t → s_{t+1}`
- وقتی مطابقت: `s_{t-1} → s_t`

### وقتی مطابقت اور جاری رکھنا

جسمانی نظاموں میں وقتی مطابقت ہوتی ہے جہاں ایکشن یا حالت میں چھوٹی تبدیلیاں نتائج میں چھوٹی تبدیلیاں پیدا کرتی ہیں۔ یہ جاری رکھنے کی خصوصیت دنیا کے ماڈل کی استحکام کے لیے ضروری ہے:

```
||f(s_t, a_t) - f(s_t + δs, a_t + δa)|| ≤ L * (||δs|| + ||δa||)
```

جہاں L ڈائی نامکس فنکشن کا لپشچ کونسٹنٹ ہے۔ دنیا کے ماڈلز کو اس جاری رکھنے کو برقرار رکھنا چاہیے تاکہ منصوبہ بندی اور سیمولیشن کے دوران بے ترتیب رویہ سے گریز کیا جا سکے۔

### جسمانی ماڈلز میں عدم یقینی کی مقدار

جسمانی دنیا کے ماڈلز کو وضاحتی طور پر عدم یقینی کی نمائندگی کرنی چاہیے جو درج ذیل سے نکلتی ہے:
1. ماڈل قریبی خامیاں
2. ماحولیاتی خلل
3. سینسر کا شور اور حالت کے تخمینے کی خامیاں
4. نامodel ڈائی نامکس

دنیا کے ماڈلز میں عدم یقینی کو یوں ظاہر کیا جا سکتا ہے:

```
s_{t+1} ~ N(μ_model(s_t, a_t), Σ_model(s_t, a_t))
```

جہاں اوسط قطعی پیش گوئی کی نمائندگی کرتا ہے اور کوویریئنس ایپسٹیمک اور الیٹورک عدم یقینی کی نمائندگی کرتا ہے۔

## ویری ایشنل دنیا کے ماڈلز

### ویری ایشنل ریکرینٹ نیٹ ورکس (VRN)

VRN دنیا کے ماڈلز کو سیکھنے کے لیے ویری ایشنل انفرینس کو ریکرینٹ نیورل نیٹ ورکس کے ساتھ جوڑتا ہے تاکہ وقتی انحیاسیت کو قبضہ کیا جا سکے:

```
z_t = encoder(s_t, z_{t-1})
z_{t+1} = recurrent_cell(z_t, a_t)
s̃_{t+1} = decoder(z_{t+1})
```

جہاں `z_t` مستقبل کی پیش گوئی کے لیے متعلقہ تاریخ کو ظاہر کرنے والی لیٹنٹ حالت کی نمائندگی کرتا ہے۔ ویری ایشنل ہدف میں شامل ہے:

```
L_VRN = E[log p(s_{t+1} | z_{t+1})] - β * D_KL(q(z_{t+1} | s_{t+1}) || p(z_{t+1} | z_t, a_t))
```

### پیش گوئی کوڈنگ نیٹ ورکس

پیش گوئی کوڈنگ سلسلہ پیش گوئی-خرابی کمی کے ذریعے دنیا کا ماڈلنگ لاگو کرتا ہے:

```
ε_bottom-up = s_t - ŝ_t
ε_top-down = z_{t-1} - ẑ_{t-1}
```

جہاں پیش گوئی کی خرابی سلسلہ سطحوں کے ذریعے پھیلتی ہے، حیرانی کو کم کرنے کے لیے اندرونی نمائندگیوں کو اپ ڈیٹ کرتے ہوئے۔

دنیا کے ماڈلز کے لیے فری انر جی پرنسپل کی فارمولیشن:

```
F = D_KL(q(s_t|ψ) || p(s_t|θ)) + E_q[log p(a_t|s_t)]
```

جہاں ψ اندرونی حالت کی نمائندگی کرتا ہے اور θ ماحولیاتی پیرامیٹر کی نمائندگی کرتا ہے۔

## ڈیفرینشیبل فزکس انضمام

### نیورل فزکس انجن

نیورل فزکس انجن جسمانی تعاملات کی سیمولیٹ کرنے کے لیے جسمانی پابندیوں کو شامل کرتے ہوئے سیکھتے ہیں:

```
f_neural(s_t, a_t, parameters) = f_known(s_t, a_t, parameters) + f_learned_corrections(s_t, a_t)
```

جہاں `f_known` معلوم فزکس کی نمائندگی کرتا ہے (مثلاً، ریجڈ باڈی ڈائی نامکس) اور `f_learned_corrections` نامodel اثرات کو قبضہ کرتا ہے جیسے فریکشن، ایئر ریزسٹنس، یا سافٹ-باڈی ڈائی نامکس۔

### ہیملٹونین نیورل نیٹ ورکس

محفوظ نظاموں کے لیے، ہیملٹونین نیورل نیٹ ورکس توانائی کی حفاظت کو برقرار رکھتے ہیں:

```
dH/dq = ∂H/∂p
dH/dp = -∂H/∂q
```

جہاں H نیورل نیٹ ورک کے ذریعے سیکھا گیا ہیملٹونین کی نمائندگی کرتا ہے، q جنرلائزڈ کوآرڈینیٹس کی نمائندگی کرتا ہے، اور p جنرلائزڈ مومنٹم کی نمائندگی کرتا ہے۔

نیورل نیٹ ورک H(q,p) کو پیرامیٹرائز کرتا ہے جبکہ ہیملٹونین ڈائی نامکس کی سیمپلیکٹک ساخت کو برقرار رکھتا ہے۔

### لاگرانجین نیورل نیٹ ورکس

پابندیوں والے نظاموں کے لیے، لاگرانجین نیٹ ورکس لاگرانجین فنکشن کو سیکھتے ہیں:

```
L(q, q̇) = T(q, q̇) - V(q)
```

جہاں T حرکی توانائی ہے اور V ممکنہ توانائی ہے۔ اولر-لاگرانجین مساوات پھر ڈائی نامکس کا تعین کرتی ہیں:

```
d/dt(∂L/∂q̇) - ∂L/∂q = Q
```

جہاں Q کنٹرول ان پٹس اور خلل کو شامل کرتے ہوئے جنرلائزڈ قوتوں کی نمائندگی کرتا ہے۔

## جسمانی نظاموں کے لیے دنیا کے ماڈل آرکیٹیکچر

### ماڈیولر دنیا کے ماڈلز

جسمانی نظاموں کو ماحولیاتی ڈائی نامکس کے مختلف پہلوؤں کو الگ کرنے والے ماڈیولر دنیا کے ماڈلز سے فائدہ ہوتا ہے:

```
World_Model = {Dynamics_Model, Contact_Model, Material_Model, Environmental_Model}
```

ہر ماڈیول کو اس کے ڈومین کے لیے مخصوص کیا جا سکتا ہے:
- Dynamics_Model: جاری حرکت اور ایکسلریشن
- Contact_Model: غیر جاری رابطے کی قوتیں اور اثرات
- Material_Model: ڈیفورمیشن اور مواد کی خصوصیات
- Environmental_Model: بیرونی خلل اور تبدیل ہوتی حالتیں

### سلسلہ دنیا کے ماڈلز

سلسلہ ماڈلز متعدد وقتی اسکیلز اور ایبسٹریکشن سطحوں پر ڈائی نامکس کو قبضہ کرتے ہیں:

```
Level_0: Low-level sensorimotor dynamics
Level_1: Object manipulation and tool use
Level_2: Navigation and spatial reasoning
Level_3: Social and task-level planning
```

ہر سطح اس کے وقتی اسکیل پر پیش گوئی فراہم کرتی ہے جبکہ اعلیٰ سطحوں سے ایبسٹریکٹ معلومات حاصل کرتی ہے۔

### متعدد سکیل وقتی انضمام

دنیا کے ماڈلز کو متعدد وقتی اسکیلز کے ذریعے معلومات کو انضام کرنا چاہیے:

```
τ_fast: Motor control (1-10ms)
τ_medium: Reactive behaviors (10-100ms)
τ_slow: Planning and adaptation (100ms-10s)
τ_very_slow: Learning and development (minutes-hours)
```

## جسمانی تعامل سے دنیا کے ماڈلز سیکھنا

### جسمانی ماحول میں خود نگران سیکھنا

دنیا کے ماڈلز جسمانی تعامل کے ڈیٹا میں ساخت سے سیکھتے ہیں بغیر بیرونی نگرانی کے:

```
L_self_supervised = Σ_t [||s_t - ŝ_t||² + ||a_t - â_t||² + ||r_t - r̂_t||²]
```

جہاں پیش گوئی ŝ_t، â_t، r̂_t گزشتہ مشاہدات اور دنیا کے ماڈل کے ذریعے پیدا کی گئیں ہیں۔

### کیوریوسٹی ڈرائیون دنیا کے ماڈل سیکھنا

ایجنٹس دلچسپی ڈرائیون ایکسپلوریشن کے ذریعے دنیا کے ماڈلز سیکھ سکتے ہیں:

```
curiosity_reward = ||prediction_error|| = ||s_t - ŝ_t||
```

یہ ذاتی حوصلہ ایجنٹ کو ایسی حالتیں ایکسپلور کرنے کے لیے ڈرتا ہے جو ماڈل کی درستگی میں بہتری لاتی ہیں۔

### ماڈل کی بہتری کے لیے فعال سیکھنا

ایجنٹ ماڈل کی عدم یقینی کو کم کرنے والے ایکشنز کو ترتیب سے منتخب کر سکتا ہے:

```
a_{t+1} = argmax_a H[p(s_{t+1}|s_t, a)] - H[p(s_{t+1}|s_t, a, D)]
```

جہاں H انتروپی کی نمائندگی کرتا ہے اور D مشاہدات کے ڈیٹا سیٹ کی نمائندگی کرتا ہے۔

## دنیا کے ماڈلز کے ساتھ ماڈل پریڈکٹو کنٹرول

### ماڈل پریڈکٹو پاتھ انٹیگرل (MPPI)

MPPI ٹریجکٹری آپٹیمائزیشن کے لیے دنیا کے ماڈلز کا استعمال کرتا ہے:

```
J(x, u) = E[Σ_t l(x_t, u_t) + φ(x_H)]
```

جہاں l اسٹیج کی قیمت ہے، φ ٹرمنل کی قیمت ہے، اور امکانات سیکھے گئے دنیا کے ماڈل کا استعمال کرتے ہوئے حساب کیے جاتے ہیں۔

### عدم یقینی کی منتقلی کے ساتھ اسٹوکاسٹک MPC

دنیا کے ماڈلز عدم یقینی کے خیال سے منصوبہ بندی کو فعال کرتے ہیں:

```
min_u E_{p(model_uncertainty)}[J(trajectory | u, initial_state)]
```

حفاظت اور کارکردگی پر امکان کی پابندیوں کے ماتحت۔

### نمونہ مبنی منصوبہ بندی

مونٹی کارلو میتھڈ دنیا کے ماڈلز کو منصوبہ بندی کے لیے استعمال کرتی ہے:

```
π(a|s) = argmax_a Σ_{i=1}^N J(trajectory_i) / N
```

جہاں ٹریجکٹریز دنیا کے ماڈل کے تقسیم سے نمونہ لی گئی ہیں۔

## دنیا کے ماڈلز کے ساتھ سیمولیشن سے حقیقت ٹرانسفر

### ماڈل سیکھنے میں ڈومین رینڈمائزیشن

مختلف ماحول میں دنیا کے ماڈلز کو تربیت دینا ٹرانسفر کو بہتر بناتا ہے:

```
p(θ) = p(θ_sim) + λ * p(θ_real)
```

جہاں θ ماڈل پیرامیٹر کی نمائندگی کرتا ہے، اور ہدف ڈومین کے فرق کے لیے مستحکم ماڈلز سیکھتا ہے۔

### سسٹم شناخت انضمام

دنیا کے ماڈلز حقیقی سسٹم پیرامیٹر کے مطابق ایڈجسٹ کرنے کے لیے سسٹم شناخت کو شامل کرتے ہیں:

```
θ_{t+1} = argmax_θ p(observations | θ) * p(θ | parameters_t)
```

### جسمانی ماڈلز میں ڈومین اڈاپٹیشن

نئے جسمانی ماحول کے لیے ماڈلز کو ایڈجسٹ کرنا:

```
L_adaptation = L_task + β * L_domain_discrepancy
```

جہاں L_domain_discrepancy ماخذ اور ہدف ڈومین کے درمیان فرق کو ماپتا ہے۔

## دنیا کے ماڈلنگ کے لیے فزکس انفارمڈ نیورل نیٹ ورکس

### تحفظ کے قانون کی پابندیاں

فزکس انفارمڈ نیٹ ورکس تحفظ کے قوانین کو شامل کرتے ہیں:

```
∇ · (ρv) = 0  (mass conservation)
d/dt(KE + PE) = P_loss  (energy conservation)
```

### توازن کی حفاظت

دنیا کے ماڈلز جیسے ترجمہ اور گردش کی عدم تبدیلی جیسے جسمانی توازن کو برقرار رکھتے ہیں:

```
f(R·s, R·a) = R·f(s,a)  (rotational symmetry)
f(s + Δ, a) = f(s,a) + Δ (translational symmetry)
```

جہاں R گردش میٹرکس کی نمائندگی کرتا ہے اور Δ ترجمہ ویکٹر کی نمائندگی کرتا ہے۔

## دنیا کے ماڈلز کے ساتھ سبب کا تجزیہ

### مداخلت کا تجزیہ

دنیا کے ماڈلز مداخلت کے ذریعے سبب کے تجزیہ کو فعال کرتے ہیں:

```
Effect of action a = E[s_{t+1} | do(a)] - E[s_{t+1} | do(a')]
```

جہاں do(a) ایکشن a لینے کی مداخلت کی نمائندگی کرتا ہے۔

### کاؤنٹر فیکچول تجزیہ

جسمانی نظام کاؤنٹر فیکچول سوالات کی حمایت کرتے ہیں:

```
"What would have happened if I had applied force F instead of F'?"
```

اس کے لیے متعدد تخیلی ٹریجکٹریز کو برقرار رکھنے کی ضرورت ہوتی ہے۔

## جسمانی دنیا کے ماڈلنگ میں چیلنج

### 1. ماڈل سیکھنے میں حقیقت کا فرق

سیمولیشن میں تربیت شدہ دنیا کے ماڈلز اکثر حقیقت میں ٹرانسفر کرنے میں ناکام ہو جاتے ہیں:

```
Gap = D(P_reality(s_{t+1}|s_t, a_t) || P_simulation(s_{t+1}|s_t, a_t))
```

جہاں D امکانات کے فرق کی پیمائش ہے۔

### 2. رابطہ اور غیر جاری رہنے کا ماڈلنگ

جسمانی رابطہ غیر جاری گی کو متعارف کراتا ہے جو ماڈل سیکھنے کے چیلنج کو جنم دیتا ہے:

```
F_contact = f(penetration_depth, relative_velocity, material_properties)
```

یہ غیر ہموار منتقلیوں کو خصوصی ماڈلنگ نقطہ نظر کی ضرورت ہوتی ہے۔

### 3. طویل افق پیش گوئی کی خامیاں

پیش گوئی کی خامیاں وقت کے ساتھ جمع ہو جاتی ہیں:

```
E[||s_t - ŝ_t||] ≤ C * λ^t
```

جہاں λ > 1 بے ترتیب نظام میں ایکسپونینشل خامی کی نمو کو پیدا کرتا ہے۔

### 4. کمپیوٹیشنل کارآمدگی بمقابلہ درستگی کے ٹریڈ آف

حقیقت کے وقت کی دنیا کے ماڈلز کو کمپیوٹیشنل کارآمدگی کے ساتھ درستگی کا توازن برقرار رکھنا چاہیے:

```
Accuracy × Computation_Time ≤ Budget
```

## جسمانی دنیا کے ماڈلز کے لیے اعلیٰ تکنیکیں

### بےزین دنیا کے ماڈلز

بےزین نقطہ نظر ماڈل پیرامیٹر کے تقسیم کو برقرار رکھتے ہیں:

```
p(θ | D) = p(D | θ) * p(θ) / p(D)
```

جہاں θ دنیا کے ماڈل پیرامیٹر کی نمائندگی کرتا ہے اور D مشاہداتی ڈیٹا کی نمائندگی کرتا ہے۔

### ایمبیل میتھڈ

متعدد ماڈلز عدم یقینی کے مختلف پہلوؤں کو قبضہ کرتے ہیں:

```
p(s_{t+1}) = Σ_i w_i * p_i(s_{t+1})
```

جہاں w_i ایمبیل وزن ہیں اور p_i انفرادی ماڈل پیش گوئی ہیں۔

### میموری اضافہ شدہ ماڈلز

بیرونی میموری سسٹم ایپی سوڈک تجربات کو ذخیرہ کرتے ہیں:

```
read_memory(state) = attention(state, memory_buffer)
write_memory(state, outcome) = update(memory_buffer, (state, outcome))
```

### جسمانی نظاموں کے لیے گراف نیورل نیٹ ورکس

گراف کی نمائندگی چیزوں کے تعلقات کو قبضہ کرتی ہے:

```
Node_features: Object properties (position, velocity, mass)
Edge_features: Relationships (distance, contact, force)
Dynamics: f(node_features, edge_features) → updated_features
```

## ہیومنوائڈ روبوٹکس میں اطلاق

### وہول بڈی ڈائی نامکس ماڈلنگ

ہیومنوائڈ روبوٹس کو فل بڈی ڈائی نامکس کے ماڈلنگ کی ضرورت ہوتی ہے:

```
M(q)q̈ + C(q, q̇)q̇ + G(q) = τ + J^T * F_external
```

جہاں M ماس میٹرکس ہے، C کوریولس قوتوں کی نمائندگی کرتا ہے، G گریویٹی ہے، اور J جیکوبین ہے۔

### توازن اور لوموکوشن پیش گوئی

دنیا کے ماڈلز چلنے کے لیے توازن کی حالت کی پیش گوئی کرتے ہیں:

```
s_balance = [CoM_position, CoM_velocity, ZMP, joint_angles, joint_velocities]
```

### مینوپولیشن اور ٹول استعمال ماڈلنگ

مینوپولیشن کے دوران چیز کے تعاملات کی پیش گوئی:

```
Object_state_{t+1} = f_hand_configuration(s_t, object_properties, contact_points)
```

## دنیا کے ماڈل کنورجنس کا ریاضیاتی تجزیہ

### استحکام کا تجزیہ

بند لوپ کنٹرول میں دنیا کے ماڈل کا استحکام:

```
V̇(x) = ∇V(x) · f(x, π_model(x)) < 0
```

جہاں V لیاپونوف فنکشن ہے اور π_model دنیا کے ماڈل کی بنیاد پر پالیسی ہے۔

### ماڈل سیکھنے کے لیے نمونہ کارکردگی

ماڈل کی درستگی کے لیے نظریاتی باؤنڈز:

```
Error ≤ O(√(d/N))
```

جہاں d ماڈل کی پیچیدگی ہے اور N تربیتی نمونوں کی تعداد ہے۔

### جنرلائزیشن باؤنڈز

نئے منظر ناموں پر ماڈل کی کارکردگی کے بارے میں باؤنڈز:

```
E[loss_on_new_env] ≤ training_loss + complexity_penalty
```

## جسمانی دنیا کے ماڈلز کے لیے جائزہ میٹرکس

### پیش گوئی کی درستگی کی میٹرکس

ماڈل کی معیار کی مقداری پیمائشیں:

#### ایک اقدام پیش گوئی کی خامی

```
MSE_single = E[||s_{t+1} - ŝ_{t+1}||²]
```

#### متعدد اقدام پیش گوئی کی خامی

```
MSE_multi = (1/H) * Σ_{h=1}^H E[||s_{t+h} - ŝ_{t+h}||²]
```

#### طویل افق کی مطابقت

```
Consistency = correlation(s_real_trajectory, s_model_trajectory)
```

### منصوبہ بندی سے متعلق میٹرکس

#### ماڈل مبنی بمقابلہ ماڈل فری کارکردگی

```
Transfer_Ratio = J_model_based / J_model_free
```

#### منصوبہ بندی کے افق کا اثر

```
Performance(horizon) = f_optimal_policy(horizon, world_model)
```

### عدم یقینی کی کیلیبریشن

#### پیش گوئی کا وقفہ کوریج

```
PICP = fraction of true values within confidence intervals
```

## جسمانی دنیا کے ماڈلنگ میں مستقبل کی سمتیں

### کوینٹم ایہانس دنیا کے ماڈلز

کوینٹم-جسمانی تعاملات کی سیمولیٹ کے لیے کوینٹم کمپیوٹنگ:

```
|ψ⟩_{t+1} = U_world(θ_params) |ψ⟩_t
```

### جماعتی دنیا کے ماڈلز

متعدد ایجنٹس کے ذریعے دنیا کے ماڈلز کا اشتراک اور بہتری:

Global_Model = f_individual_models(agent_1, agent_2, ..., agent_N)
```

### لائف لانگ دنیا کے ماڈل سیکھنا

ایسے ماڈلز جو جاری طور پر ایڈجسٹ اور بہتری کرتے ہیں:

Model_{t+1} = update(Model_t, new_experience, task_changing_indicators)
```
